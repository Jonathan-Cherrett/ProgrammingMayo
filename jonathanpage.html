Computer Graphics and OpenGL

The Open Graphics Library (Or OpenGL) is a cross language, cross platform API

It is used to interact with Graphics Processing Units (GPUs) to render 2D and 3D graphics

  • 1991: Silicon Graphics Inc Begin developing OpenGL
  • 1992: OpenGL 1.0 released on 30th June
  • 2006: The Non-Profit consortium Knronos Group begin managing OpenGL's development
  • Updated and extended to include more features and make use of developments in GPU technology
  • Current version: 4.6, Released July 31st 2017
